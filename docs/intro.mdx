# 开始游玩

欢迎来到 **苍岚屿** 空岛生存，本页面将引导你如何进入服务器

## 😊快速开始

### 第一步：下载客户端
:::info
如果您有自己的客户端，可以跳过此步骤，注意需要1.21.11版本
:::
1. 加入 [QQ群(点击)](https://qm.qq.com/q/BFRlcXDsiY)
2. 下载群文件中的客户端文件（.zip）
3. 解压到桌面

### 第二步：启动游戏

1. 打开 **启动器** 字样的文件
2. 输入你的游戏ID，可随意（不能中文）
3. 点击启游戏
4. 完成！
:::warning
游戏ID不支持中文，使用中文ID将无法进入服务器！
:::

### 第三步：连接到服务器
1. 启动完成后，点击**多人游戏**
2. 添加服务器
3. 输入服务器IP：`mc.geyino.top`
4. 进入服务器
5. 完成！

### 第四步：创建角色
首次进入服务器，你需要：
1. 按照提示进行注册
2. 完成！

:::warning
如果第一次进入服务器提示你需要 **登录**，则说明你的id被其他玩家注册了。
你需要更改id后再次进入服务器
:::
