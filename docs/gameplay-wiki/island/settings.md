# 岛屿基础设置

岛屿基础设置用于调整岛屿的各种行为规则，包括时间、天气、生物生成、爆炸、液体流动等内容。

| 设置项 | 说明 |
|--------|------|
| 永远白天 | 岛屿时间始终保持白天。不能与其他时间设置同时启用。 |
| 永远正午 | 岛屿时间始终保持正午。不能与其他时间设置同时启用。 |
| 永远夜晚 | 岛屿时间始终保持夜晚。不能与其他时间设置同时启用。 |
| 永远午夜 | 岛屿时间始终保持午夜。不能与其他时间设置同时启用。 |
| 永远下雨 | 岛屿天气始终保持下雨。不能与其他天气设置同时启用。 |
| 永远晴天 | 岛屿天气始终保持晴天。不能与其他天气设置同时启用。 |
| 苦力怕爆炸 | 是否允许苦力怕爆炸破坏方块。 |
| 作物生长 | 是否允许农作物正常生长。 |
| 鸡下蛋 | 是否允许鸡正常下蛋。 |
| 末影人搬运方块 | 是否允许末影人拾取和搬运方块。 |
| 火焰蔓延 | 是否允许火焰向周围方块蔓延。 |
| 恶魂火球爆炸 | 是否允许恶魂火球破坏方块。 |
| 岩浆流动 | 是否允许岩浆流动。 |
| 动物自然生成 | 是否允许动物自然生成。 |
| 怪物自然生成 | 是否允许怪物自然生成。 |
| 玩家对战 | 是否允许玩家之间进行 PvP。开启时，所有访客会被传送至主城，以防止恶意困人。 |
| 刷怪笼生成动物 | 是否允许刷怪笼生成动物。 |
| 刷怪笼生成怪物 | 是否允许刷怪笼生成怪物。 |
| TNT 爆炸 | 是否允许 TNT 爆炸破坏方块。 |
| 树木生长 | 是否允许树木自然生长。 |
| 水流流动 | 是否允许水流流动。 |
| 凋灵爆炸 | 是否允许凋灵爆炸破坏方块。 |
| 死亡不掉落 | 是否启用死亡不掉落。开启后，玩家在岛屿内死亡不会掉落物品。 |
