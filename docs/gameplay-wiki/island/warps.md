# 岛屿传送点

岛屿传送点用于给岛屿设置多个可公开或内部使用的传送位置。   


## 岛屿内传送点
:::info
岛屿内传送点默认为私有，不对公众开放  
可在控制面板调整`/island warp`
:::
| 命令 | 说明 |
| :--- | :--- |
| `/island warp` | 管理您岛屿的传送点 |


也可以在告示牌上写上`[岛屿传送点]`来设置岛屿的传送点


## 公共岛屿传送点
| 命令 | 说明 |
| :--- | :--- |
| `/island warps` | 查看全服的岛屿传送点 |

## 设置访客传送点

:::info
访客传送点默认不存在
:::
告示牌上写上`[欢迎]`即可设置访客传送点的地点
