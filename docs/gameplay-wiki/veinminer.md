# 连锁挖掘
:::tip
此页面正在完善中
:::
:::info
如果你觉得某些常用方块不能被连锁，可以向管理员建议添加
:::

连锁挖矿功能可以让你一次性挖掉一整片相连的方块，比如整条矿脉或整棵树，省时又省力！


## 如何使用

你只需要**按住潜行键（默认Shift）**，然后用正确的工具破坏方块，即可触发连锁挖矿了

:::note
如果你安装了配套的客户端模组，还可以设置自己喜欢的快捷键，不用一直按着 Shift。   
配套的模组可以在 **[QQ群](https://qm.qq.com/q/BFRlcXDsiY)** 中的群文件内下载并安装，目前仅支持Fabric
:::
