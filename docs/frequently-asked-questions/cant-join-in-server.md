# 无法进入服务器
:::tip
此页面正在完善中
:::

## 服务器已关闭
服务器已关闭连接。可进入QQ群获取服务器的最新消息

## 连接超时
服务器无法找到玩家的连接。

## 身份验证服务器目前处于宕机状态。请稍后再试。抱歉！
身份验证服务器不可用。（微软的服务器炸了）

## 你已被此服务器封禁
被服务器封禁，可联系管理员以解除封禁（一般情况下不会封禁玩家）

## 聊天消息验证失败 
无法验证聊天消息来源。

## 非法聊天字符
客户端被拒绝发送特定字符，如 `§` 符号。

## 客户端不兼容！请使用 xxx 版本
客户端版本与服务器不兼容。目前服务器使用 `1.21.11` 版本，不支持其他版本

## invalid packets
服务器插件的漏洞。若出现请联系管理员处理

## 登录超时
登录过程总共超过了 `500` 秒。请尝试重新连接
