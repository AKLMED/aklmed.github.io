# 🏝️ 开始游玩

欢迎来到 苍岚屿 空岛生存！本页面将一步一步引导你进入这片充满挑战与和谐的岛屿世界。

---

## 🚀 快速上手四步走

### 📥 第一步：下载客户端

:::tip
如果你已有 1.21.11 版本客户端  
可以跳过此步骤，直接前往【第三步】连接服务器。
:::

1. 加入我们的 QQ 交流群 👉 [点击加入](https://qm.qq.com/q/BFRlcXDsiY)
2. 在群文件中找到并下载最新的客户端压缩包（.zip）
3. 将压缩包解压到你的电脑桌面（建议新建一个文件夹）

---

### ▶️ 第二步：启动游戏

1. 在解压后的文件夹中，双击打开名为 启动器 的应用程序
2. 在启动器界面输入你的游戏 ID
3. 点击 启动游戏 按钮

:::warning[关于游戏 ID]
不支持中文，请使用英文字母或数字组合。  
如果使用中文ID，将直接导致无法进入服务器！
:::

---

### 🔗 第三步：连接服务器

1. 进入游戏主界面，点击 多人游戏
2. 点击 添加服务器 按钮
3. 在服务器地址栏输入：`mc.geyino.top`
4. 点击 完成，然后双击刚刚添加的服务器即可进入！

---

### ✨ 第四步：创建角色

首次进入服务器，一切都非常丝滑：
1. 进入后请根据屏幕聊天栏的提示完成注册流程
2. 注册成功后，你就可以正式开始你的空岛生存之旅啦！

:::warning
如果刚进服务器，系统提示你需要**登录**，说明你当前使用的 ID 已经被其他玩家注册过了。  
解决方法：退出游戏，回到【第二步】换一个从来没被注册过的 ID，再次进入即可。
:::
